# College Website Project

This project is a **College Website** built with modern web development technologies, featuring a **full responsive frontend**, an **admin dashboard**, and a robust **backend**.

## Features

### Frontend
- Built using **React.js** and **CSS3** for a responsive and interactive user interface.
- Fully **responsive admin dashboard** for managing college data.
- API calls handled using **Axios** for smooth client-server communication.

### Backend
- Built using **Node.js** with **Express.js** following the **MVC (Model-View-Controller) pattern**.
- Implements **JWT authentication** for secure access.
- **CRUD operations** for managing data.
- File uploads handled with **Multer**.
- Connected to **MongoDB Atlas** for database storage.

### Other Features
- Modern and clean UI for both students and admin.
- Secure and structured backend architecture.
- Fully functional with real-time data management.

## Tech Stack
- **Frontend:** React.js, CSS3, Axios  
- **Backend:** Node.js, Express.js, JWT, REST API, Multer, MVC Pattern  
- **Database:** MongoDB Atlas
